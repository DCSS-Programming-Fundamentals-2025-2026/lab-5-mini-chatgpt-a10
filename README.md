Lib.ChatConsole (Група А10)

Опис :
Цей компонент відповідає за реалізацію інтерактивного консольного інтерфейсу (REPL) для взаємодії користувача з моделями генерації тексту Mini-ChatGPT.

Головна мета компонента - приймати текстовий ввід від користувача, обробляти службові команди для керування параметрами генерації (семплінгу), та передавати запити (промпти) до мовної моделі.





Структура:
"ChatRepl" - головний клас, який запускає безкінечний цикл (Read-Eval-Print Loop).
"ReplOptions" - зберігає поточні налаштування чату (Temperature, Top-K, Seed, MaxTokens).
"CommandRegistry" - відповідає за реєстрацію та розпізнавання команд.





Команди:
"/temp \[значення]" - змінити температуру генерації (наприклад: /temp 0.5).
"/topk \[значення]" - змінити параметр Top-K (наприклад: /topk 5).
"/seed \[значення/null]" - зафіксувати або скинути seed для випадковості.
"/reset" - скинути налаштування генерації до стандартних та очистити екран.
"/help" - вивести список усіх доступних команд.
"/quit" - вийти з програми.





Інтеграція (Етап 2):

1. Створити клас, який реалізує ITextGenerator (він має поєднати навчену нейромережу та алгоритми семплінгу від А5).
2. Створити екземпляр нашого головного класу ChatRepl, передавши йому цей готовий генератор у конструктор: ChatRepl repl = new ChatRepl(generator);
3. Викликати метод запуску циклу з базовими налаштуваннями: repl.Run(temperature, topK, seed);
4. Подбати про обробку системного потоку введення, хоча наш REPL вже має вбудований базовий захист від сигналу EOF (NullReferenceException).
